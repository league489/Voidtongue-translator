## What is Voidtongue?
Voidtongue is an artificial language originating from video game called Warframe. In the game lore it is spoken mostly by The Man in the Wall, mysterious entity inhabiting parallel dimension called Void. The language  posses limited syntax ( few prefixes, suffixes, conjunctions and pronouns) and vocabulary (only 38 words with known meaning). The purpose of this project is to extend it by defining most possible new words and grammar rules, consisting on official data, that would allow for decently accurate , both - directional translation .
